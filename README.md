# Westshore Watch Firmware

Firmware for Westshore Watch X1 drone detection nodes.

## Variants

- **[`WSW-Firmware/`](./WSW-Firmware/)** — Custom Westshore PCB (ESP32-C5 and ESP32-C6 targets, single codebase, target selected via `idf.py set-target`)
- **[`XIAO-C5-Firmware/`](./XIAO-C5-Firmware/)** — Seeed Studio XIAO ESP32-C5 dev board variant (lower-cost alternative SKU)

Each variant folder contains its own README with build, flash, and hardware notes.
